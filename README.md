This project is a Library Management System made using Python.
It helps to manage books in a simple way like adding books, issuing them to students, and returning them.

I used a dictionary to store all the book records.
Each book name is used as a key, and its details like availability, student name, issue date, and number of days are stored as values.

The system has basic features:

Add a new book
Show all books
Issue a book to a student
Return a book

When a book is issued, the system stores the student name and current date.
While returning, it checks how many days the book was used.

If the book is returned late, a fine is calculated:

First week → ₹10 per day
Second week → ₹20 per day
Third week → ₹30 per day

The fine increases week by week.

How to use dictionary and nested dictionary

Basic logic building in Python
